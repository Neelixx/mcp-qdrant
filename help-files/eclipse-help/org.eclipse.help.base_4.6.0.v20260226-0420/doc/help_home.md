# Using the help system

Browse topics in the **Contents** frame (![Contents icon](contents_view.svg)) on the left. Click on a topic to have it displayed. Use the **Back** and **Forward** buttons to navigate within the history of viewed topics. 

## Searching

To quickly locate topics on a particular subject in the documentation, enter a query in the **Search** field. Use the **Search** frame (![Search icon](search_results_view.svg)) to display the Search view. You can narrow the **scope** of your search by selecting only the sections you are interested in. 

## Synchronizing

Clicking the **Show in Table of Contents** button (![Show in Table of Contents icon](e_synch_toc_nav.svg)) will select that topic in the navigation tree. The **Link with Contents** button (![Link with Contents icon](e_auto_synch_toc.svg)) keeps the navigation tree synchronized to the current topic. 

## Capabilities

To show documentation about capabilities that are disabled in the application, select the **Show All Topics** button (![Refresh / Show Current Topic icon](e_show_all.svg)) . This affects the table of contents and search results. 

## Keyword Index

To show an alphabetically sorted index of keywords select the **Index Frame** (![Index icon](e_index_view.svg)). Not all products contain an index.
